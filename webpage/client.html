Welcome to the Streamlit Video Client App for Group 12

This is a webpage for the final project of Group 12 for the course ITCS 6166/8166 (Computer Communication and Networks) under the guidance of Dr. Pu Wang. This link has detailed instructions on how to run the project using Google Colab. We recreated the results of the project HumanNeRF: Free-viewpoint Rendering of Moving People from Monocular Video (Weng et al., 2022).

This simple webpage connects to a Streamlit server that hosts a video. The video is served using Flask to ensure smooth streaming.

How it works:

  1. The Streamlit server displays the video using the Streamlit library.
  2. The Flask server serves the video content as a stream.
  3. This HTML page uses the <video> tag to display and control the video.

Click play and enjoy the video!
